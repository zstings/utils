## random :tada: :100: 
均衡获取指定范围的随机整数
::: info
返回一个min 和 max之间的随机整数。如果你没有参数，那么将返回0和1之间的整数。如果你只传递一个参数，那么将返回0和这个参数之间的整数。
:::
#### 参数 
- **min** `number = 0` 范围最小整数
 
- **max?** `number` 范围最大整数
 
#### td.ts
::: info
`random(min?, max?): number`
:::
#### 返回 
- `number` 
::: tip
随机整数
:::
#### 实例 
均衡获取0或者1的数


```ts
random()
```
均衡获取0或者5的数


```ts
random(5)
```
均衡获取1或者10的数


```ts
random(1, 10)
```
